# Dungeon-Game
I was bored so I made a small console game.

Controls: <br>

Simply type in console: <br>
right - move one space to the right <br>
left - move one space to the left <br>
up - move one space up <br>
down - move one space down <br>

@ - Player <br>
# - Wall <br>
% - Door <br>
